# Lab01
## Программа для решения трех задач линейного алгоритма (Вариант 8):
1.  **Перевод скорости** из км/ч в м/с.
2.  **Перевод скорости** из м/с в км/ч.
3.  **Вычисление пути**, пройденного за время *t* при скорости *v* (км/ч), с выводом результата в метрах и километрах.

### Входные данные:
- Для задачи 1: Вещественное неотрицательное число (скорость в км/ч).
- Для задачи 2: Вещественное неотрицательное число (скорость в м/с).
- Для задачи 3: Два вещественных неотрицательных числа (скорость в км/ч и время в часах).

### Как запустить через Visual Studio
1.  Склонируйте репозиторий.
2.  Откройте файл решения `Lab01.sln`.
3.  Запустите программу (нажмите `Ctrl+F5` для запуска без отладки).

### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub.
- В терминале введите команду: git clone <ссылка на репозиторий>
После выполнения команды в текущей папке появится папка с проектом.

### Особенности программы
Меню для выбора задач: Программа предлагает пользователю выбрать одну из трех задач или выйти.
Полная валидация ввода:
- Проверка на пустой ввод.
- Провеhка, является ли введенное значение числом (double.TryParse).
- Проверка на неотрицательность введенных чисел (скорость и время не могут быть отрицательными).
- Проверка корректности выбора пункта меню.
Модульная структура: Код организован в отдельные методы (Task1, Task2, Task3) для каждой задачи и вспомогательный метод GetPositiveDouble для повторного использования логики ввода.
Форматированный вывод: Все результаты выводятся с двумя знаками после запятой ({variable:F2}).
